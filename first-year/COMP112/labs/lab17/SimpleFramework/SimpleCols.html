SimpleCols: A (very) Basic CSS Framework

Overview

SimpleCSS was adapted from material published on w3schools.com. It is a very basic framework that provides functionality for columnised layouts.

Usage

To Use SimpleCols have SimpleCols.css available and include the following in the head section of your HTML document:

<link rel="stylesheet" type="text/css" href="simpleCols.css">

Simple Columns

Columns are wrapped in rows. Each column unit is one twelth the width of its parent element. The number of column units in a row must always add up to 12. See below for examples.

<div class="row">
    <div class="col-3">...</div>
    <div class="col-9">...</div>
</div>

The number of columns add up to 12; col-3 + col-9.
